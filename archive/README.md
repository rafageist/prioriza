# Archive

This directory keeps historical and source material that is not the canonical
editable Quarto manuscript.

- `source-docx/` stores the latest DOCX source used for the Quarto conversion.
- `legacy-docs/` stores the earlier Markdown/Pandoc manuscript and PDF build
  files preserved for traceability.

The canonical editable source for the current manuscript is the Quarto book at
the repository root, starting with `_quarto.yml` and `index.qmd`.
