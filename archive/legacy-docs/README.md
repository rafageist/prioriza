# Legacy Documentation

This directory preserves the pre-Quarto manuscript and build files.

- `docs-v0.1/`: earlier Markdown chapters.
- `pdf-order-v0.1.txt`: chapter order used by the previous Pandoc PDF build.
- `build_pdf-v0.1.sh`: previous Pandoc/LaTeX PDF build script.
- `thesis-v0.1/`: earlier thesis workspace placeholder.

These files are historical reference material. They are no longer the canonical
editable manuscript.
