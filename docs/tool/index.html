Prioriza en el navegador

Esta ruta queda reservada para una futura herramienta estática que permita definir elementos, aspectos, niveles y explicaciones sin depender de un servidor.

Alcance previsto

  • Definir elementos y aspectos.
  • Registrar valores y reglas de nivelación.
  • Declarar niveles de prioridad de los aspectos.
  • Calcular y explicar la estructura de prioridad.
  • Exportar datos en formatos interoperables.

La lógica de cálculo no está implementada todavía. Esta página no es una calculadora y no debe usarse para tomar decisiones.
