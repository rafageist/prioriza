Herramienta Prioriza

Define elementos, aspectos, valores y niveles de prioridad. El cálculo se ejecuta en el navegador. Los datos se guardan automáticamente en el almacenamiento local.

Demo experimental. Esta herramienta es una demostración del método. No debe usarse para tomar decisiones reales sin validación independiente.

Aún no hay tablas. Crea una nueva para comenzar.

Cada aspecto asigna niveles locales desde sus valores: la función define qué valores reciben prioridad; el NPA (nivel de prioridad del aspecto) indica su peso relativo.
